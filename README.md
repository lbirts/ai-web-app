## Getting Started

git clone `<my-repo>`
cd ai-chat-app
npm install

# Create .env.local with:

# OPENAI_API_KEY=your_key_here

# Optional for free local testing (no OpenAI charges):

# MOCK_OPENAI=true

npm run dev

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
